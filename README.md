Quod
==========================

Spring MVC web module to search for Financial statements.

Used APIs,

- Java 8
- Spring MVC 3.2.8
- Java iText 2.1.7
- Java poi 3.17
- Spring Security 3.2.3
- HTML5, CSS3, Boostrap 3, JQuery 2

You can visit the live version via [Quod](http://quod-dev-alpha.herokuapp.com//).
